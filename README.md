# vimrc-cpp
VIM　C/C++　集成开发环境

## 使用说明

下载该工程　
```shell
git clone https://github.com/alvisisme/vimrc-cpp.git ~/.vim_runtime
```

安装脚本
```shell
cd ~/.vim_runtime
sh install_vimrc.sh
```

VIM命令模式下执行`:PlugInstall`完成插件安装

插件安装于 **～/.vim/plugged/** 目录下
