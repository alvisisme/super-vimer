# vimrc-cpp
VIM　C/C++　集成开发环境

## 使用说明

下载该工程　
```shell
git clone https://github.com/alvisisme/vimrc-cpp.git
```

安装脚本
```shell
cd vimrc-cpp
sh install_vimrc.sh
```

VIM命令模式下执行`:PlugInstall`完成插件安装

插件安装于**～/.vim/plugged/**目录下

## 插件列表

* [vim-plug](https://github.com/junegunn/vim-plug)

插件管理

* [NERDTree](https://github.com/scrooloose/nerdtree)

目录树显示
